Comando: gestion de dataset/snapshot (create/rename/delete/mount/key)
PATH remoto usado:

Argumentos y plantillas:
- Crear dataset/volumen/snapshot:
  - zfs create [flags] [props] <dataset>
  - zfs snapshot [flags] [props] <dataset@snap>
- Renombrar: zfs rename <old> <new>
- Borrar: zfs destroy [-f|-r|-R] <dataset|dataset@snap>
- Montar/desmontar:
  - zfs mount <dataset>
  - zfs umount <dataset> / zfs unmount <dataset> (Windows)
  - recursive umount script
- Claves:
  - zfs load-key <dataset>
  - zfs unload-key <dataset>
  - zfs change-key -o keylocation=prompt [-o keyformat=passphrase] <dataset>
