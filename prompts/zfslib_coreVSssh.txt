  Las únicas operaciones que usan zfslib_core (con fallback a SSH)

  ┌──────────────────┬──────────────────────────────┐
  │    Operación     │        Función libzfs        │

  ├──────────────────┼──────────────────────────────┤
  │ Unmount          │ localLibzfsUnmountDataset()  │
  ├──────────────────┼──────────────────────────────┤
  │ Rename           │ localLibzfsRenameDataset()   │
  ├──────────────────┼──────────────────────────────┤
  │ Set property     │ localLibzfsSetProperty()     │
  ├──────────────────┼──────────────────────────────┤
  │ Inherit property │ localLibzfsInheritProperty() │
  └──────────────────┴──────────────────────────────┘

  Y solo aplican cuando: conexión local + no Windows + detectLocalLibzfs() devuelve true. En cualquier otro caso van directamente por SSH.
