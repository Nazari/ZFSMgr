# Quick manual

ZFSMgr manages connections and ZFS actions.

- Connections: create/edit connections and refresh status.
- Datasets: source/target actions (Copy, Level, Sync).
- Advanced: actions on one dataset (Breakdown, Assemble, From Dir, To Dir).
- Pools: import/export, status and properties.
- Logs: execution details and application status.

Check "Shortcuts and states" for action enabling rules.
