# 上下文菜单

GUI 在两个区域使用右键菜单。

## `连接` 区域右键菜单

在当前选中的连接上：

- `刷新`
- `编辑`
- `删除`

说明：

- `Local` 以及重定向到 `Local` 的连接，`编辑` 与 `删除` 会禁用。
- 有操作运行时，`刷新` 会被阻止。

## `内容 <pool>` 区域右键菜单

在当前选中的数据集/快照上：

- `Rollback`
- `创建`
- `删除`
- `源`
- `目标`
- `拆分`
- `合并`
- `来自目录`
- `到目录`

## 规则

- 破坏性操作必须二次确认。
- 菜单启用/禁用与其它操作采用相同校验逻辑。
- 操作执行期间，不安全选项会被禁用。
