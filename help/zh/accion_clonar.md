# 操作：克隆

目标：使用 `zfs clone` 将快照克隆到目标数据集。

按钮可用条件：

- 源必须是 `快照`。
- 目标必须是 `数据集`（不能选择快照）。
- 源和目标必须在同一连接中。
- 源和目标必须属于同一个池。

克隆窗口可选参数：

- `-p` 如果父数据集不存在则自动创建。
- `-u` 不自动挂载克隆结果。
- `-o property=value`（每行一条）用于设置克隆属性。

基础命令：

`zfs clone [-p] [-u] [-o property=value]... <source@snapshot> <target_dataset>`

说明：

- 不满足条件时，按钮会保持禁用。
- 操作完成后会刷新目标连接及其数据集视图。
