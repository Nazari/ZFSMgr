# 操作：复制

目标：从源端发送快照，在目标端接收。

条件：

- 源：已选择快照。
- 目标：已选择数据集。
- 源和目标都必须使用 OpenZFS `2.3.3` 或更高版本。

行为：

- 使用 `zfs send` 与 `zfs recv`。
- 如果源与目标是两个不同的远程 SSH 连接，ZFSMgr 会尽量让传输直接从源端到目标端进行。
  这种情况下，数据流不会经过运行 ZFSMgr 的机器；该机器只负责控制与进度显示。
- 在组合日志中显示进度。
- 该动作会先加入 `待执行更改`，只有在应用待执行更改时才真正执行。
- 完成后刷新受影响的连接/树。
- 若任一端低于 `2.3.3`，该操作会被阻止。
