# 快速手册

ZFSMgr 用于管理连接和 ZFS 操作。

- 左侧面板：
- `连接`：简单列表（每行一个连接）。
- `新建`：新建连接或新建池。
- `操作`：传输与高级操作。
- 右侧面板：
- 选中连接时：显示 `连接 <名称>` 标签页。
- 对有可见池的连接：每个已导入/可导入池显示一个标签页。
- `属性 <pool>`：池属性（上）与池状态（下）。
- `内容 <pool>`：数据集内容（上）与数据集属性（下）。
- 不可导入池不会生成标签页，会在连接状态中显示原因。
- 日志：单一 `组合日志` 面板（包含带连接前缀的 SSH/PSRP 输出）。

请查看“快捷键与状态”了解按钮启用条件。
